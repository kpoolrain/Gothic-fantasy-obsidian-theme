## 哥特魔幻风obsidian主题
### 引言：
一个自制的主题分享
本主题采用哥特式魔幻渐变风格，营造一股旧式，黑幻的氛围，推荐使用深色模式。
全方面进行了样式调整，甚至添加有一些小小的功能。

### 安装
1. 下载主题文件，并存放在一个文件夹里
2. 将文件放入Obsidian的`.obsidian/themes`文件夹
3. 在Obsidian设置 > 外观 > 主题 中选择"主题"
4. 或者你也可以单独将css放在css片段里。具体启用为：你的仓库位置/.obsidian/snippets/主题文件夹，打开obsidian设置——外观——css片段，启用。

也可以安装Durwent字体，进一步加强哥特风。
